# resin_art_ecommerce
An online e-commerce website built using PHP and MySQL to sell and showcase resin art products.  It allows users to browse, add to cart, buy products, and provides an admin dashboard for managing products.

1. Homepage 
- Displays the landing page of the Resin Art E-commerce platform.
- Highlights featured resin artworks, navigation bar, and user-friendly layout.
- Serves as the entry point for browsing and purchasing resin art.
![image alt](https://github.com/GanyaPoojary/resin_art_ecommerce/blob/main/resin_art_homepage.png)
2. View Resin Art 
- Shows the product detail page for a selected resin artwork.
- Includes artwork image, description, price, and "Add to Cart" option.
- Provides users with detailed information before making a purchase.
![image alt](https://github.com/GanyaPoojary/resin_art_ecommerce/blob/main/view_resin_art.png)
3. Buy Resin Art 
- Demonstrates the checkout process.
- Users can confirm selected items, enter payment details, and place orders.
- Ensures a smooth and secure buying experience.
![image alt](https://github.com/GanyaPoojary/resin_art_ecommerce/blob/main/buy_resin_art.png)
4. Admin Dashboard 
- Interface for administrators to manage the platform.
- Provides insights into orders, users, and product listings.
- Helps admins maintain and monitor the e-commerce system efficiently
![image alt](https://github.com/GanyaPoojary/resin_art_ecommerce/blob/main/admin_dashboard.png)
5. Manage Users 
- Admin functionality for managing registered users.
- Includes options to view, edit, or remove users.
- Ensures proper control and security of the platform.
![image alt](https://github.com/GanyaPoojary/resin_art_ecommerce/blob/main/manage-user_resin.png)
